444状态码

适用于屏蔽非安全请求或DDOS防御

1、nginx server配置

server {
    listen 8089;
    location /123 {
            proxy_pass http://192.168.1.145:8080;
    }
    location / {
            index html/index.html;
    }

    location /abc {
            return 444;
    }
}
